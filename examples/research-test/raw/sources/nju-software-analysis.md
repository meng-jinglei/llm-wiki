# 南京大学《软件分析》课程

| 讲次 | 主题 |
|------|------|
| 1 | Introduction |
| 2 | Intermediate Representation |
| 3 | Data Flow Analysis — Foundations I |
| 4 | Data Flow Analysis — Foundations II |
| 5 | Data Flow Analysis — Applications |
| 6 | Interprocedural Analysis |
