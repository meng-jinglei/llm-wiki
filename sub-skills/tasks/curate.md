## curate（整理）

当用户希望进行人工主导的重组或优化时使用。

示例：
- 合并重复页面
- 拆分过大的页面
- 重命名页面
- 重新组织主题结构
- 重写摘要或概述页面
- 刷新用户指出错误或过时的页面

### 步骤

1. 读取目标页面及附近的上下文。
2. 如果结构性变更较大，先解释建议的变更。
3. 应用最小的连贯编辑集。
4. 按需更新 `index.md` 和链接。
5. 在 `log.md` 中追加一条整理或修正记录。
6. 报告所有触及的路径。

### 输出结构

```
<修改的页面>                ← 合并/拆分/重命名后的页面
index.md                    ← 链接更新（如有必要）
log.md                      ← curation 记录
```
